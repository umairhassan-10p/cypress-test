# Getting Started

Take clone of the UI tests from the repository
```
$ git clone 
```

### Prerequisites

- NodeJS
- Npm

### Installation

Installs all the packages present in `package.json`
```
$ npm install
```
## Running the tests

Goto project folder and run UI tests via Cypress test runner, use following command,
```
npx cypress open
```
